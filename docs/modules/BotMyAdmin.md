# BotMyAdmin

BotMyAdmin 数据库管理


## Exec <Badge type="tip" text="开发者" vertical="top" />

执行无返回值 sql

命令格式：

```text
/bma exec <sql>
```


## Open <Badge type="tip" text="开发者" vertical="top" />

连接数据库

命令格式：

```text
/bma open <db_name>
```


## Query <Badge type="tip" text="开发者" vertical="top" />

执行查询 sql

命令格式：

```text
/bma q <sql>
```

# 关于 AimuBot

AimuBot 为 [InariAimu](https://github.com/InariAimu) 开发的一款 IM 机器人，设计初衷只是一个猫猫图机器人（？）。

## 关于赞助

AimuBot 的主要支出费用为 服务器月租 和 ~~大量猫粮~~。

**AimuBot 既不会收费，也不会接受任何形式的赞助。**

能够得到用户的喜爱和支持，就是对 AimuBot 最大的赞助。
